Labeling Widget

Configured via JavaScript

labels is the list of draggable labels; text_lines is the list of lines to label; correct_labels maps each line index to a list of correct label indices (multiple labels per line are allowed).

Configured via HTML

Write a two-column table inside a div.marimo-labeling: column 1 is the text to label and column 2 is the correct label name (or a comma-separated list of names for lines that accept multiple labels). The available label list is derived automatically from the unique names in column 2.
